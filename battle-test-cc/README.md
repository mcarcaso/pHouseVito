# Battle Test - CC Harness Web App

A professional static web application for the CC Harness battle testing environment.

## Features

- **Live Timestamp**: Real-time updating timestamp that refreshes every second
- **Modern Design**: Professional gradient backgrounds with glass morphism effects
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Elements**: Hover effects and smooth animations
- **Performance Optimized**: Lightweight static files with no build process required
- **Developer Friendly**: Console logging and performance monitoring

## Files Structure

```
battle-test-cc/
├── index.html          # Main HTML file
├── styles.css          # Modern CSS styling
├── script.js           # JavaScript for live timestamp and interactions
└── README.md           # This documentation file
```

## Usage

1. Open `index.html` in any modern web browser
2. The timestamp will automatically start updating
3. No server or build process required - pure static files

## Features in Detail

### Live Timestamp
- Updates every second with full date and time
- Includes timezone information
- Formatted in a user-friendly way (e.g., "Sunday, February 16, 2026 at 02:30:45 PM PST")

### Modern Styling
- Gradient backgrounds with animated color shifts
- Glass morphism design with backdrop blur effects
- Smooth hover animations and transitions
- Professional color scheme and typography

### Responsive Design
- Mobile-first approach
- Breakpoints at 768px and 480px
- Optimized for all screen sizes

### Interactive Elements
- Click effects on timestamp
- Parallax scrolling effects
- Smooth page load animation
- Console welcome message for developers

## Browser Compatibility

Works in all modern browsers:
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## Performance

- Minimal JavaScript footprint
- CSS animations use GPU acceleration
- Performance monitoring in console
- Optimized for 60fps animations

## Customization

You can easily customize:
- Colors in `styles.css` (look for gradient definitions)
- Timestamp format in `script.js` (modify the `options` object)
- Animation timings (adjust CSS transition and animation durations)
- Content in `index.html`

## Development

This is a static web app requiring no build tools or dependencies. Simply edit the files and refresh the browser to see changes.
